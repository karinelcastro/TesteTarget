3) Descubra a lógica e complete o próximo elemento:
a) 1, 3, 5, 7, ___

Essa sequência mostra os numeros ímpares, logo o próximo elemento é 9.

b) 2, 4, 8, 16, 32, 64, ____

O próximo número é sempre o dobro do anterior, portanto o proximo elemento seria 128.

c) 0, 1, 4, 9, 16, 25, 36, ____

Essa sequência apresenta quadrado dos números naturais. O próximo elemento seria 49.

d) 4, 16, 36, 64, ____

Essa sequência mostra o quadrado de cada um dos números pares,a partir do número 2. O próximo elemento seria 100.

e) 1, 1, 2, 3, 5, 8, ____

Essa é a sequência de Fibonacci, onde cada número é a soma dos dois anteriores. O próximo elemento será 13.

f) 2, 10, 12, 16, 17, 18, 19, ____

Aparentemente a sequência apresenta os números iniciados pela letra "D". O próximo número seria o 200.
